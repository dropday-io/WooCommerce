Automate dropship orders by email, FTP, EDI or API to your suppliers. Seamless order synchronization for your webshop.

== Description ==

Dropday automates your dropshipping workflow by automatically sending orders to your suppliers via email, FTP, EDI, or API calls.

**Features:**

* Automatic order synchronization with suppliers
* Multiple delivery methods: Email, FTP, EDI, API
* Easy configuration through WordPress admin
* Real-time order status updates
* Support for multiple suppliers

== Installation ==

1. Download the Dropday plugin
2. Go to the admin area of your WordPress webshop
3. Navigate to 'Plugins' → 'Add New' → 'Upload Plugin'
4. Select the .zip file and click 'Install Now'
5. Click 'Activate Plugin'

== Configuration ==

1. Go to Settings → Dropday Order Sync
2. Enter your API key and Account ID from your Dropday Dashboard
3. Enable 'Live mode' after testing
